Russische Grammatik: Possessivadjektive

Kurzanleitung

Bildung basierend auf dem Genitiv Singular:

  • Genitiv auf -А → -ОВ- (Иван → Ивана → Иванов)
  • Genitiv auf -Я (unbetont) → -ЕВ- (Андрей → Андрея → Андреев)
  • Genitiv auf -Я (betont) → -ЁВ- (Соловей → Соловья → Соловьёв)
